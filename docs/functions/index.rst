Functions
=========

Functions are the building blocks for customizable slots in DynamicGui.
Below are the built-in functions for DynamicGui, addons may add more.
If you are interested in making functions take a look at the `developer docs <../functionapi>`_.

* checklevel
* executec
* executep
* gui
* nopermission
* pay
* permission
* pmsg
* removeslot
* send
* broadcast
* setdata
* setenchants
* setlore
* setname
* settype
* sound
* statistic
